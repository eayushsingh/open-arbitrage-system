# Open Arbitrage System 🚀

A realtime crypto arbitrage dashboard built using:

- React + Vite
- FastAPI
- WebSockets
- Render + Vercel deployment

## Features

- Live arbitrage opportunity dashboard
- Realtime WebSocket updates
- Spread and profit visualization
- Filtering and sorting
- Responsive UI
